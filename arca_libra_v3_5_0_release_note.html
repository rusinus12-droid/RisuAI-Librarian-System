Arca Release Note

LIBRA_World_manager V3.5.0.js

V3.0.0 기준에서 V3.5.0으로 넘어오며 달라진 핵심 기능과, 지금 버전 기준으로 실제 체감되는 변경점을 보기 좋게 정리한 안내문입니다.

Release Focus
장기 컨텍스트 통합 강화
Core Theme
분석 · 재분석 · 병합 안정화
UI / UX
대시보드 · 모바일 · 설정 개선
한 줄 요약
V3.5.0은 메모리만 저장하는 보조 플러그인 수준을 넘어서, 메모리 / 엔티티 / 관계 / 세계관 / 내러티브 / 수동 로어북 / 세션 전환 / 재분석 / 대시보드를 하나의 파이프라인으로 묶은 통합형 장기 컨텍스트 매니저에 가까운 버전입니다.

1. 메모리 엔진과 턴 처리 강화

  • 턴 리커버리, pending finalize, refresh checkpoint 복구, cold start 검증 흐름이 정리되었습니다.
  • 메모리 재분석이 추가되어 누락된 기억을 다시 스캔하고, 후검증을 거쳐 보충 저장할 수 있습니다.
  • 메모리 재분석 중 새로 채택된 후보끼리도 중복 비교가 들어가도록 보강되어, 인접 턴 유사 기억 중복이 줄었습니다.
  • 메모리 검색의 auto 가중치가 웹소설/장르문학 톤을 더 잘 반영하도록 확장되었습니다.

2. 엔티티 / 관계 / 내러티브 관리 고도화

  • 엔티티, 관계, 내러티브에 수동 보호가 추가되어 GUI에서 직접 저장한 값이 자동 수정에 쉽게 덮이지 않습니다.
  • 보호 설정 / 보호 해제를 GUI에서 바로 전환할 수 있습니다.
  • 엔티티 / 내러티브 재분석 버튼이 추가되어 현재 채팅 로그 기준으로 다시 구조화할 수 있습니다.
  • 성관념 / 성적 취향 / 직업 / 위치 같은 필드가 성격 특성에 뭉쳐 들어가지 않도록 더 잘 분리됩니다.

3. 세계관 시스템 전면 확장

  • 세계관이 단순 텍스트 뭉치가 아니라 exists / systems / physics / custom 구조로 더 안정적으로 분배됩니다.
  • 전역 세계 특성이 정리되었고, 복합 세계 신호를 더 넓게 자동 감지합니다.
  • 장면용 세계관 보정이 추가되어, 현재 장면에서 어떤 세계관 요소를 우선 해석해야 하는지 보조 프롬프트로 반영합니다.
  • 세계관 재분석은 이제 전역 특성까지 normal turn 처리에 가깝게 복원합니다.
  • 웹소설 계열 장르 키워드와 퓨전 장르 판정이 강화되어 현대/판타지/헌터/아카데미/선협/사이버펑크 쪽 해석 폭이 넓어졌습니다.

4. 과거 대화 분석 / 재분석 / Hypa 병합 개선

  • 과거 대화 분석, 과거 대화 재분석, Hypa V3, 수동 로어북 모두 누적 병합 + 검증 흐름으로 정리되었습니다.
  • 분석/재분석은 이제 단순 요약이 아니라, 엔티티 / 관계 / 세계관 / 내러티브 / 전역 세계 특성 / 장면용 세계관 보정까지 더 폭넓게 반영합니다.
  • 하이파와 수동 로어북은 원본도 보존하고, 구조 데이터 병합 결과도 함께 남깁니다.

5. 수동 로어북 기능 추가

  • 수동 로어북 추가 기능이 생겨, 손요약/설정/정리 메모를 `lmai_user_N` 형태로 직접 저장할 수 있습니다.
  • 저장 번호를 직접 지정할 수 있고, 기존 번호를 덮어쓸 때는 확인창이 뜹니다.
  • 원본은 최상위 참고 로어북으로 보존되며, 저장 직후 메인 LLM 분석을 거쳐 LIBRA 구조 데이터에도 병합됩니다.

6. 세션 전환과 상태 복구 강화

  • 대화 이어가기가 구조 데이터, 메모리뿐 아니라 수동 로어북, Hypa 원본 소스, 채팅 노트까지 더 잘 계승합니다.
  • 그리팅 격리 로직도 세션 전환 후 새 채팅 기준으로 다시 맞춰져, 첫 assistant-only 인사 격리 흐름이 더 안정적입니다.
  • 페이지 새로고침이나 저장 타이밍 어긋남에 대비해 checkpoint / recovery / pending stabilization 흐름이 계속 보강되었습니다.

7. GUI / 대시보드 / 설정 개선

  • 작업 진행 상황을 보여주는 대시보드가 추가되어, 토큰/호출/진행률/백그라운드 작업 상태를 시각적으로 확인할 수 있습니다.
  • 과거 대화 분석 / 재분석, 하이파, 수동 로어북, 메모리 / 엔티티 / 세계관 / 내러티브 재분석도 내부 단계별 퍼센트가 더 자연스럽게 오르도록 보강되었습니다.
  • 설정은 이제 파일 기반 export/import 방식으로 관리됩니다.
  • 모바일 대응, 다크 모드 대응, 중앙 정렬, 카드형 정보 표시 등 GUI 사용성이 개선되었습니다.

8. 호환성과 보조 옵션

  • 유저 무시 금지 옵션이 추가되어, 켜면 현재 유저 입력을 최우선 응답 축으로 둡니다.
  • 라이트보드 삽화 태그 호환 옵션이 추가되어, 삽화 모듈의 `<lb-xnai>...</lb-xnai>` 태그를 보존할 수 있습니다.
  • NSFW 지침 전달 옵션이 추가되어, 켰을 때만 Director / Story Author에 NSFW guidance가 전달됩니다.
  • NSFW가 꺼져 있으면 이 추가 문구들은 전달되지 않고, 기존 Director / Story Author 설계 그대로 동작합니다.

정리

V3.0.0이 기본적인 메모리/구조화 플러그인에 가까웠다면, V3.5.0은 장기 서사 유지, 수동 개입 보호, 분석 결과 검증 병합, 세계관 해석 보조, 세션 전환 복원, 가시적인 작업 대시보드까지 포함한 실사용 중심 버전입니다.
